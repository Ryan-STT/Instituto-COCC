CÓDIGO

O COCC (Centro de Oportunidade Cultural à Criança) é um projeto desenvolvido em Java com o propósito de criar um site para crianças de 5 a 12 anos com conteúdo cultural de músicas, livros, desenhos, jogos etc. O projeto é estruturado em HTML5, CSS3, Java e JavaScript com DataBase MySQL
